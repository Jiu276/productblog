## 下载步骤

1. **访问网站**：点击上面的链接
2. **搜索图片**：使用建议的关键词
3. **选择图片**：选择高质量、合适尺寸的图片
4. **下载图片**：
   - 主图片建议下载 1200x600 或更大尺寸
   - 头像建议下载 200x200 或更大
5. **重命名文件**：按照下面的命名规范

## 文件命名规范

保存到 `static-site/assets/images/` 目录：

```
iPhone 相关：
- iphone-main.jpg (主图片)
- iphone-detail.jpg (详情图片)

MacBook 相关：
- macbook-main.jpg (主图片) 
- macbook-detail.jpg (详情图片)

Apple Watch 相关：
- watch-main.jpg (主图片)
- watch-detail.jpg (详情图片)

AirPods 相关：
- airpods-main.jpg (主图片)
- airpods-detail.jpg (详情图片)

作者头像：
- author-john.jpg (男性作者)
- author-lisa.jpg (女性作者)
- author-mike.jpg (其他作者)
```

## 图片要求

- **格式**：JPG 或 PNG
- **主图片尺寸**：900x400 或相近比例
- **头像尺寸**：100x100 或相近比例  
- **质量**：高清，无水印
- **内容**：产品清晰可见，背景简洁
